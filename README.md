VPP-MVP (Virtual Protest Protocol)
VPP is an online civic participation platform designed to enable safe, anonymous, and large-scale public demonstrations in digital space.

The goal is to provide a democratic infrastructure where citizens can:

Join demonstrations anonymously
Support, oppose, or observe an issue
Participate without exposing personal information
Reduce barriers to civic engagement
This repository contains the MVP implementation of the VPP concept.

Status: MVP Prototype
License: MIT
Live Demo:https://will-of-america.net/
Why VPP?
Many citizens, especially young people, hesitate to join physical demonstrations because of social pressure, employment concerns, or privacy risks.

VPP aims to solve this problem by creating a digital alternative to traditional public demonstrations.

Current Features
Implemented:

User registration with anonymous handles
Avatar-based participation
Yes / No / Observe participation model
Demonstration creation
Demonstration browsing
In Progress:

Real-time participant counter
Moderation tools for organizers
Mobile responsive UI
Planned:

End-to-end encryption for votes
Blockchain timestamping for transparency
Multilingual support (EN/JP)
Integration with existing petition platforms
Origin
This project was created in Japan following observations of declining civic participation among younger generations.

The author believes democratic participation should be possible even when physical attendance is difficult or risky.

VPP is not a replacement for street protest. It's a gateway for people who would otherwise stay silent.
